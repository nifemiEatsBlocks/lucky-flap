Lucky Flap:
A chaotic twist on flappy bird

About the Game
This is a twist on the classic flappy bird, adding new features and a chaotic lucky dice-roll mechanic. 

 Features:
1. Dynamic Pipe Types
The pipes are no longer simple static obstacles.

Pipes that oscillate up and down

Pipes with varying space between them

Pipes that become invisible

2. The Luck Mechanic:
At key moments, you'll roll a dice for a  buff or debuff. 

Examples:

Smaller Bird: Your hitbox shrinks, making it easier to dodge.

Ghost Mode: Become intangible and pass right through pipes.

Gravity Flip: The world turns upside down!

...and more!
